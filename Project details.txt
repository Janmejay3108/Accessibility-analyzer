- Created a platform that analyses websites for accessibility (WCAG 2.1 and ADA compliance) issues, provides AI-powered remediation suggestions, bug fixes, meanwhile tracking progress over time, storing user data and analysis history.
- Additional Goal: Include a feature for developers to request custom website or app development services from the platform.
- Technologies used: React.js, JavaScript, Firebase, Express.js, Node.js, axe-core, Playwright, Lighthouse.
